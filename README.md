# single-spa-angular

## start microfrontends

### terminal 1
```bash
cd mf-root
npm install
npm start
```

### terminal 2
```bash
cd ng-mf-header
npm install
npm run serve:single-spa:ng-mf-header
```

### terminal 3
```bash
cd ng-mf-app
npm install
npm run serve:single-spa:ng-mf-app
```

[http://localhost:9000](http://localhost:9000)
